# Student Performance Prediction System

This project predicts whether a student will pass using machine learning.

## Features

* Data preprocessing using Polars
* Feature engineering
* Model training (Logistic, RandomForest, XGBoost)
* Model evaluation
* Hyperparameter tuning
* Feature importance visualization

## Structure

* data/ → dataset
* notebooks/ → ML notebook
* models/ → saved model

## How to Run

1. Open notebooks/main.ipynb
2. Run all cells

## Output

* Accuracy, Precision, Recall
* Best model selection
